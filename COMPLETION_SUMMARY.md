## 🎮 How to Use

### Local Development
```bash
# Start server
python -m http.server 8000

# Open browser
# http://localhost:8000

# Login with demo
Email: demo@pulse.com
Password: demo123
```

### Deployment
```bash
# Vercel (easiest)
vercel deploy --prod

# Netlify
git push origin main

# GitHub Pages
git push origin main --force
```

### Configuration
1. Update `supabase.js` with your credentials
2. Create database from `DATABASE_SCHEMA.md`
3. Upload audio files to storage
4. Customize colors in `style.css`
